Welcome To Seat Selector



This is a Final Year Honours Project, created by Isaac Ponsford, student reference number: 201458733



In this project I have created a program which simulates on optimal loading of an airplane. It does this by loading plane and passenger CSV files into respective database tables

These database tables interact with a "blackbox" rules engine which assigns the passengers into the plane seats, using upgrades and downupgrades it ensures that the maximum number of seats are occupied, that upgrades happen whenever possible and the rule engine powers an intuative UI which combines it all
